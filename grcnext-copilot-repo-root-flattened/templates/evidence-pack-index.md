This index lists the evidence supporting an assessment or a Service
Dossier. Each row is an evidence item. The full field set for each item
is defined in
[`../schemas/evidence-item.schema.json`](../schemas/evidence-item.schema.json)
and the working ledger is
[`evidence-ledger.csv`](evidence-ledger.csv). Prefer evidence of actual
execution over statements of intended procedure. Do not store secrets or
protected personal data in the pack. Redact with neutral labels.

## How to use this index

1. Give every evidence item a stable Evidence ID.
2. Record the claim it supports, its type, owner, date or period,
   verification status and freshness.
3. Note any contradiction or limitation.
4. For provisional items, record the evidence-conversion action that
   would make them verified.
